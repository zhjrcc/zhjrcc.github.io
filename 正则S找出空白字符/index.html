/\s+/g和/\s/g的区别

First published 2022-06-12
Modified 2022-06-12

正则表达式/\s+/g和/\s/g目的都是为了找出字符串中所有的空白字符。

但两者有什么区别呢 ？

看一段代码：

let name = 'zhao   zi  long';
let name_1 = name.replace(/\s/g, ''); // zhaozilong
let name_2 = name.replace(/\s+/g, ''); // zhaozilong

咋看之下，似乎没区别，再看一段代码:

let name = 'zhao   zi  long';
let name_1 = name.replace(/\s/g, '#'); // zhao###zi##long
let name_2 = name.replace(/\s+/g, '#'); // zhao#zi#long

现在我们清楚了，\s+实际上从遇到空白字符开始一直会匹配，直到匹配不上为止就把这一段替换成了1个#。而\s是遇到一个空白字符立刻就替换成1个#。
