Start-0

First published 2023-03-08
Modified 2023-03-08

Self-discipline计划完全失败，还是败在自己内心不够坚定。
再来，这次好好来。
不要内耗，不要选择，只要当前最重要，首要完成的。
鱼与熊掌不可兼得，记住了，任何东西，只要不是必须要做的，都可以不要去想也不要去做，只要专注你当下必须要完成的。
本来你主要是要完成A的，但做A的途中，你想B，想C，脑子想了，自然而然就会干扰你的A事件。最后你就又要想要做B，而真正的A却被你放置脑后了。
这是总结出来的经验了，必须清楚明白你当前真正想做的事情，然后专注进去就行了，任何其他事件ABCDEFG绝对就是干扰因素。
干扰因素越少，脑子就越清晰，产出质量就越高。
杀死干扰因素，现在我认为，只要不是主要事件，任何其他冒出来的都是干扰因素。
尽量保证让自己脑子清晰简单认定要做的事情，而不是让它三心二意，让脑子一团糟。

Start-0就是新的计划了，早睡早起计划。
每次觉得自己睡眠不够不够，那你打游戏，看电影熬夜的时候怎么又能熬夜呢？
到了要学习工作了你就怕自己睡眠不够了？
压根就没有的事，只是你自己就怕了，害怕累。
但这点累你都受不了吗？
况且，下班回来立刻洗澡睡觉，早上5点起床，这样有什么累？
耐心点，总会变好的，努力尽力去试试吧，日积月累，事情一定会回到正轨的。
就这样，Start-X开始了。
杀死干扰因素，保持最主要的事件，专心下去。
把握早上的时间而不是熬夜熬出时间。
