React + Vite

  1. npm create vite@latest,填入项目名，选择框架，选择TS还是JS，cd react-project npm install npm run dev
  2. 将src文件夹下除了main.jsx的文件都删除了，以便我们学习如何自己编写组件。

JSX写组件

  1. main.jsx是react整个文件的入口。创建App.jsx作为应用的根组件。UserProfile.jsx是用户信息展示组件，UserEmail.jsx，UserName.jsx，UserFavoriteAnimate.jsx分别是用户信息的一些组件
  2. 开始分别写这几个组件
  3. 下载’prop-types’,对每个组件中的props都进行验证
  4. 写样式，可以写常规的css，也可以将每个css作为一个模块，将css文件命名为，style.module.css，通过`import styles from ‘./style.module.css’，将jsx的className={style.className}
  5. 渲染列表，使用数组的map,需要对列表添加一个key属性。
