盒模型

将页面的盒模型设为 border-box

:root {
  box-sizing: border-box;
}

*,
::before,
::after {
  box-sizing: inherit;
}

负外边距

正的外边距会将容器的边缘往里推，而负的外边距则会将边缘往外拉

控制溢出行为

overflow：visible hidden scroll auto。
visible,默认值，内容可见，就算溢出容器。
hidden，隐藏溢出的内容。
scroll，显示滚动条。
auto，如果内容溢出显示滚动条。
